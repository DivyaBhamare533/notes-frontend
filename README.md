# 📝 Notes Frontend

A responsive and user-friendly Notes web application built using HTML, CSS, and JavaScript. The project provides a clean interface for creating and managing notes in the browser and demonstrates core front-end web development skills.

---

## 📌 Project Overview

This project is a lightweight notes application designed to help users organize their ideas and tasks. It focuses on creating an intuitive user experience using modern HTML, CSS, and JavaScript without requiring a backend.

---

## ✨ Features

- Create new notes
- Edit existing notes
- Delete notes
- Responsive user interface
- Clean and modern design
- Fast and lightweight

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
